## Run Locally   

Clone the project

``` console
git clone git@github.com:Harrytimbog/NODEJS-AUTHENTICATION.git
```

Go to the project directory

``` console
cd NODEJS-AUTHENTICATION
```
Install NPM Packages / dependences

``` console
npm install
```

Start Server

``` console
npm start
```
